Moderación

  • /ban

    Banea a un usuario del servidor de forma permanente

  • /unban

    Desbanea a un usuario del servidor mediante su ID

  • /kick

    Expulsa a un usuario del servidor temporalmente

  • /clear

    Borra mensajes en masa de un canal específico

  • /timeout

    Aisla a un usuario temporalmente.

  • /vip

    Otorga o quita manualmente el rol de VIP a un usuario

Información

  • /userinfo

    Muestra información detallada de un usuario del servidor

    Muestra información completa sobre nuestro servidor

    Muestra los comandos disponibles del servidor
